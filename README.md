MIC AND FRAC

El proposito del proyecto es combinar la libreria de audio que nos ofrece p5.js para analizar el micro para poder de alguna forma modificar una figura fractal que se presentara en pantalla. En este caso he implementado lo que seria una espiral fractal, no una cola de dragon, en este caso se puede ver como del centro aparecen como unos brazos que parecen petalos.

En este caso al pillar el micro sucede diferentes cosas con la figura fractal, la primera seria que la figura se empieza como un ventilador, después tambien los colores cambian, los petalos se abren y finalmente la figura se agrandece, todo esto sucede dependiendo del ruido que capte. Si capta mucho ruido habra mucho caos (colores cambiantes, petalos abiertos...) pero si se hace poco ruido los cambios a la figura seran ligeros. Para poder saber el nivel de ruido que estamos causando, a la derecha he puesto un medidor de ruido.

La idea de la figura es que al hacer mucho ruido los petalos se abriesen y al cruzar las lineas entre si se creasen unas figuras en el medio. También me hizo gracia el hecho de mover la figura como un ventilador ya que lo veia muy estatico.

El uso de IA se ha utilizado para analizar mejor el audio.
